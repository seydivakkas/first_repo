# WeaveVision Public Textile Benchmark

Bu dal, `IndustrialTextileDataset/type1cam1` üzerinde normal-only TextureMemoryModel eğitir ve kalibre eder. Ardından aynı dondurulmuş modeli RAW-FABRID'in MVTec biçimli açık kaynak alt kümesinde, yeniden eşik ayarlamadan değerlendirir.

Üretilen artifact:

- `benchmark_report.json`
- `BENCHMARK_SUMMARY.md`
- `predictions.csv`
- `public_proxy_report.html`
- `public_proxy_texture_memory.npz`
- örnek inceleme görüntüleri

Claim seviyesi yalnızca `OPEN_SOURCE_TEXTILE_PROXY_ONLY`'dir. Sonuçlar Merinos veya başka bir şirket üretim hattı doğrulaması değildir.
